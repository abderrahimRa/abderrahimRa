## Featured Project

# OptiGra — Web-Based Scheduling Optimization Platform

**Status:** Private repository / public demo available  
**Live Demo:** [OptiGra Demo](PASTE_OPTIGRA_LIVE_URL_HERE)  
**Repository:** Private  
**Type:** Master's thesis project  

OptiGra is a web-based decision-support platform for **production scheduling optimization**. It transforms scheduling theory, especially **Graham's alpha | beta | gamma notation**, into a guided workflow where users define the problem, run compatible solving methods, and analyze the result through schedules, Gantt charts, and performance indicators.

### What It Does

- Converts scheduling theory into a guided web workflow
- Uses Graham's `alpha | beta | gamma` notation for problem definition
- Supports multiple scheduling problem classes
- Provides rules, heuristics, and metaheuristic solving logic
- Includes a Simulated Annealing solver for complex scheduling problems
- Displays schedules using Gantt charts and performance indicators
- Connects a modern frontend with a backend optimization engine

### Tech Stack

`Next.js` `React` `TypeScript` `Tailwind CSS` `shadcn/ui` `FastAPI` `Python` `Firebase` `Scheduling Algorithms` `Simulated Annealing`

### Why It Matters

OptiGra connects **industrial engineering theory** with a practical software interface. Instead of keeping scheduling methods as abstract formulas, it turns them into a usable decision-support workflow for production planning.
