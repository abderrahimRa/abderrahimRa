## <img src="https://raw.githubusercontent.com/tandpfun/skill-icons/main/icons/NextJS-Dark.svg" width="28" height="28" alt="Next.js icon" /> Main Project: OptiGra

<div align="center">

**OptiGra** is my master’s thesis project: a web-based decision-support platform for production scheduling. It transforms Graham’s `alpha | beta | gamma` notation into an interactive workflow where users can define scheduling problems, run suitable methods, and analyze schedules through Gantt charts and performance indicators.
