## 🎯 Featured Project: OptiGra

**Master's thesis project: Web-Based Scheduling Optimization Platform**

<table>
  <tr>
    <td bgcolor="#F8FAFC">
      <strong>OptiGra</strong> — Production Scheduling Decision-Support System<br/>
      <span style="font-size:12px; color:#666;">A web platform that transforms industrial scheduling theory into interactive workflows for production planning optimization.</span>
      <br/><br/>
      
      <strong>What It Does:</strong>
      <ul>
        <li>Converts <strong>Graham's α|β|γ notation</strong> into guided web workflows</li>
        <li>Supports multiple scheduling problem classes and constraints</li>
        <li>Provides rules, heuristics, and metaheuristic solving logic</li>
        <li>Includes <strong>Simulated Annealing solver</strong> for complex optimization</li>
        <li>Visualizes schedules with <strong>Gantt charts</strong> and performance indicators</li>
        <li>Real-time decision support for industrial planning</li>
      </ul>
