OptiGra — Web-Based Scheduling Optimization Platform

OptiGra is my master’s thesis project: a web-based decision-support platform for production scheduling.

It transforms Graham’s alpha | beta | gamma notation into an interactive workflow where users can define scheduling problems, run suitable methods, and analyze schedules through Gantt charts and performance indicators.

What OptiGra does:

    Converts scheduling theory into a guided web workflow

    Supports multiple scheduling problem classes

    Uses Graham’s alpha | beta | gamma notation for problem definition

    Provides rules, heuristics, and metaheuristic solving logic

    Includes a Simulated Annealing solver for complex scheduling problems

    Displays schedules with Gantt charts and performance indicators

    Connects a modern Next.js frontend with a FastAPI optimization backend
